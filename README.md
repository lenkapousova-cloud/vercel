# Akční Ceny — Nasazení na Vercel

## 1. Nahrajte na GitHub
Nahrajte celou složku `akcni-ceny-app` na GitHub jako nový repozitář.

## 2. Nasaďte na Vercel
1. Jděte na vercel.com → New Project
2. Importujte GitHub repozitář
3. Framework: Vite
4. Přidejte Environment Variables:
   - VITE_SUPABASE_URL = https://eifooaghbprllczieowj.supabase.co
   - VITE_SUPABASE_ANON_KEY = (váš anon key ze Supabase)
5. Deploy!
